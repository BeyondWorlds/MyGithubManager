# github-android-collect
收集一些用的最多的android库

## 简介
希望github出个star分组编辑功能；面对自己成百上千的star项目，确实不太好管理。这个项目主要用来收集我自己常用的android库，一般来说都是stars>3000的项目，并且做了个简要的分类，尽量避免相同功能的库。
